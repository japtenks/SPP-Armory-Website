# Guild Recruitment Thread Example

This is the target format for guild recruitment threads generated by the site.

## Post 1: Recruitment Tagline

Title:

```text
<Darkside of Initium> is Recruiting!
```

Body:

```text
[b]<Darkside of Initium>[/b] is now open for recruitment!

Led by [b]Rinerine[/b], we are looking for dedicated adventurers to join our ranks.

Contact [b]Rinerine[/b] in-game to apply.
```

## Notes

- Post 1 stays the recruitment pitch.
- Post 2 is the living roster post and should be updated in place.
- Welcome, departure, and call-to-action lines may rotate between approved stock phrases.
- The roster should always remain easy to scan first by names and levels.
